### Ex. 1
The data set [6, 19, 33, 33, 39, 41, 41, 43, 51, 57] has median Q₂ = 40, first quartile Q₁ = 33, and third quartile Q₃ = 43. An outlier in a data set is a value that is more than 1.5 times the interquartile range below the first quartile (Q₁) or more than 1.5 times the interquartile range above the third quartile (Q₃), where the interquartile range is defined as Q₃ − Q₁.
How many outliers does this data set have?

- (A) 0
- (B) 1
- (C) 2
- (D) 3
- (E) 4

### Ex. 2

A sixth number is added to the list. This sixth number is greater than *x*.

2. The mean of the six numbers is greater than *x*.
3. The range of the six numbers is greater than *z*.

- (A) none of them
- (B) 1 only
- (C) 2 only
- (D) 3 only
- (E) 1 and 2 only
- (F) 1 and 3 only
- (G) 2 and 3 only
- (H) 1, 2 and 3

- (C) 16½
- (D) 16⅔
